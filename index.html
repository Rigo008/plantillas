TIPOS DE RAM

Memoria Ram

La memoria de acceso aleatorio (Random Access Memory, RAM) se utiliza como memoria de trabajo de computadoras y otros dispositivos para el sistema operativo, los programas y la mayor parte del software. En la RAM se cargan todas las instrucciones que ejecuta la unidad central de procesamiento (CPU) y otras unidades del computador, además de contener los datos que manipulan los distintos programas.

Static RAM (SRAM)

Es una memoria de acceso aleatorio (RAM) que retiene bits de datos en su memoria mientras se suministra energía . A diferencia de la RAM dinámica (DRAM), que almacena bits en celdas que constan de un condensador y un transistor, la SRAM no tiene que actualizarse periódicamente.

Dynamic RAM (DRAM)

Se utiliza ampliamente como memoria principal de una computadora . Cada celda de memoria DRAM está compuesta por un transistor y un capacitor dentro de un circuito integrado, y un bit de datos se almacena en el capacitor.

Synchronous Dynamic RAM (SDRAM)

Es un nombre genérico para varios tipos de memoria dinámica de acceso aleatorio (DRAM) que se sincronizan con la velocidad de reloj para la que está optimizado el microprocesador. Esto tiende a incrementar el número de instrucciones que el procesador puede realizar en un tiempo determinado.
